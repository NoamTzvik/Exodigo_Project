# Field Operations Checklist: Pittsburgh Project

## 1. Pre-Scan Phase (Preparation)
- [ ] Conduct preliminary research on site history and utility records.
- [ ] Confirm team availability (4 operators).
- [ ] Inventory equipment: Check IDS, EM, and all 15 transmitters.
- [ ] Verify commute route (20 mins to AB&B).
- [ ] Plan initial 35x40m grid with 2m overlap.
- [ ] Check weather forecast (Confirm Wed snow probability).

## 2. Scan Days (Execution)
- [ ] **Monday:** Attend 09:00 AM meeting. Set up primary baseline.
- [ ] **Tuesday:** Attend 07:30 AM meeting. Execute Day 2 polygons.
- [ ] **Wednesday:** Monitor snow. Ensure equipment is covered. Resume PM.
- [ ] **Thursday:** Attend 07:30 AM meeting. Execute Day 4 polygons.
- [ ] **Friday:** Final scan audit. 
- [ ] **Daily:** Mark "Key Points" in the field using physical markers.
- [ ] **Daily:** Take photos of the terrain, incline, and equipment setup.
- [ ] **Daily:** Verify overlap quality between adjacent polygons.

## 3. Post-Scan Phase (Finalization)
- [ ] Upload all sensor data to the Israel mapping team.
- [ ] Complete "Key Point" report with GPS coordinates.
- [ ] Compile all field photos into the final report.
- [ ] Pack and clean equipment.
- [ ] Final team debrief and documentation sign-off.
